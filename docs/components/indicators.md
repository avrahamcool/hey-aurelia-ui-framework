#### `ui-chip`
Value chips

* Attributes
  * `label = ''`
  * `glyph = ''`
  * `theme = ''`
  * `width = ''`: Set minimum width
  * `small/normal/large`: Sizes
  * `removable`

* Event
  * `remove.trigger`

---

#### `ui-breadcrumb`
Breadcrumbs

* Attributes
  * `theme = ''`
  * `class = 'ui-themed'`: Set solid color theme

* Events
  * `change.trigger`


#### `ui-crumb`
Breadcrumbs node

* Attributes
  * `id = ''`
  * `href = ''`: Can also have `href` to navigate views using breadcrumbs

* Events
  * `click.trigger`
