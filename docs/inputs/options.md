#### `ui-option-group`

* Attributes
  * `vertical`: Align option inputs vertically
  * `cols = ''`: Group option inputs into columns
  * `name = ''`: Apply name to all radio inputs
  * `value.bind = ''`: Radio input selected value

* Events
  * `change.trigger = ''`: Value change event
