#### Basic Typography Classes

###### Font Family
* `ui-font-normal`
* `ui-font-serif`
* `ui-font-input`
* `ui-font-mono`

###### Font Styles
* `ui-bold`: bold weight
* `ui-strong`: medium weight
* `ui-italic`
* `ui-uppercase`
* `ui-lowercase`
* `ui-titlecase`
* `ui-small-caps`
* `ui-ellipsis`
* `ui-selectable`: enable text selection

###### Font Sizes
* `ui-font-small`
* `ui-font-big`
* `ui-font-large`

###### Alignment
* `ui-text-start`
* `ui-text-end`
* `ui-text-center`
* `ui-text-justify`

###### Display Styles
* `ui-display-1`
* `ui-display-2`
* `ui-display-3`
