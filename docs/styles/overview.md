#### Basic Helper Classes

###### Padding
* `ui-pad-none`
* `ui-pad-all`
* `ui-pad-h`: horizontal left/right
* `ui-pad-v`: vertical top/bottom

###### Margin
* `ui-margin-none`
* `ui-margin-all`
* `ui-margin-h`
* `ui-margin-v`

###### Borders
* `ui-border-none`
* `ui-border-all`
* `ui-border-h`
* `ui-border-v`
* `ui-border-s`: start
* `ui-border-e`: end
* `ui-border-t`: top
* `ui-border-b`: bottom

###### Visibility
* `ui-hide`: display none
* `ui-hidden`: visibility hidden

###### Color Classes
* `ui-bg-{theme-color}`
* `ui-text-{theme-color}`
* `ui-border-{theme-color}`
* `ui-hover-{theme-color}`: apply theme color for link hover

* Theme colors
  * `white`
  * `gray`
  * `black`
  * `light`
  * `dark`
  * `primary`
  * `secondary`
  * `info`
  * `danger`
  * `success`
  * `warning`
